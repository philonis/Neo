# 网络搜索

## 快速开始

```python
# 搜索示例
result = web_search(query="今天天气", max_results=3)
```

## 工作流程

### 操作列表

| 操作 | 说明 | 参数 |
|------|------|------|
| search | 执行搜索 | query, max_results |
| news | 搜索新闻 | query, max_results |

### 详细步骤

1. **解析查询**: 理解用户搜索意图
2. **执行搜索**: 调用搜索引擎API
3. **解析结果**: 提取标题、摘要、链接
4. **返回结果**: 格式化输出

## 使用原则

- **优先使用**: 当需要实时信息时，优先使用搜索而非浏览器
- **效率优先**: 搜索比浏览器更轻量，适合获取公开信息
- **结果处理**: 返回的结果包含标题、链接、摘要

## 注意事项

- 搜索结果可能包含广告
- 部分内容可能需要登录才能查看
- 对于复杂页面，可能需要使用 browser_agent
